
# Генератор Дипломной Работы с LM Studio

Скрипт автоматизирует процесс написания дипломной работы, используя локальную языковую модель (LM Studio). Он позволяет задать тему и дополнительные требования, после чего модель формирует детальный план, генерирует структуру глав и пишет содержательный текст для каждой главы.

## ✨ Возможности
- **Локальная работа:** Полная приватность данных — всё происходит на вашем компьютере через LM Studio.
- **Планирование:** Автоматическая генерация структуры диплома в формате JSON.
- **Генерация глав:** Пошаговое создание текста для каждой главы с учетом подпунктов.
- **Форматирование:** Поддержка Markdown (заголовки, жирный текст) и автоматическое преобразование в документ `.docx`.
- **Таблицы и структуры:** Умное распознавание таблиц в тексте модели и их корректная отрисовка в Word.

## 🛠️ Требования

Для запуска скрипта необходимо установить следующие зависимости:

```bash
pip install python-docx requests
```

> **Примечание:** Скрипт использует `urllib.request`, поэтому дополнительного пакета для HTTP-запросов не требуется, но наличие Python 3.8+ рекомендуется.

## 🚀 Установка и запуск

1. Убедитесь, что на вашем компьютере запущена локальная версия **LM Studio** (порт по умолчанию: `http://localhost:1234`).
2. Клонируйте или скачайте скрипт в папку.
3. В терминале перейдите в директорию со скриптом и выполните:

```bash
python generate_diploma.py
```

## 📝 Использование

После запуска вы увидите приглашение для ввода данных:

1. **Тема:** Введите тему вашей дипломной работы (например, *"Разработка системы умного дома"*).
2. **Дополнительные правки:** Укажите любые специфические требования к стилю или содержанию (например, *"Использовать много формул"* или *"Акцент на Python"*).

Скрипт автоматически проверит связь с LM Studio и начнёт генерацию. В процессе работы будут выводиться статусы выполнения глав.

## 📂 Структура выходных данных
После завершения скрипт создаст файл `Дипломная_работа.docx`, содержащий готовый шаблонный документ с оформленными заголовками, подпунктами и текстом.

## ⚙️ Настройка LM Studio
Убедитесь, что в настройках LM Studio (вкладка "Server") порт установлен на `1234`. Если вы используете другой порт или IP-адрес (например, при удаленном доступе), измените переменную `LM_STUDIO_URL` в начале скрипта:

```python
LM_STUDIO_URL = "http://YourIpAddress:1234/v1/chat/completions"
```

## 🛡️ Обработка ошибок
- Если связь с LM Studio не устанавливается, скрипт выдаст ошибку и завершится.
- При ошибках парсинга JSON модель попытается извлечь данные вручную из блока кода (markdown code block).
- Код работает не идеально, проверяйте сохраненый файл на наличие ошибок
